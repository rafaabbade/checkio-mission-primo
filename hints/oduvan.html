Qual pode ser o primeiro passo para resolver esse problema?

Primeiro precisamos determinar as características de um número primo:

    • Verifique se o número é menor ou igual a 1. Se for, ele não é primo.
      Verifique se o número é igual a 2. Se for, ele é primo.
      Verifique se o número é par. Se for, ele não é primo.
      Para cada número ímpar entre 3 e a raiz quadrada do número, verifique se o número é divisível por esse número. Se for, ele não é primo.
      Se o número passou em todas as etapas anteriores, ele é primo.
