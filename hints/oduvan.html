Qual pode ser o primeiro passo para resolver esse problema?

Primeiro precisamos determinar as características de um número primo:

  1. Verifique se o número é menor ou igual a 1. Se for, ele não é primo.
  2. Verifique se o número é igual a 2. Se for, ele é primo.
  3. Verifique se o número é par. Se for, ele não é primo.
  4. Para cada número entre 3 e o número, verifique se o número é divisível por esse número menos 1. Se for, ele não é primo.
  5. Se o número passou em todas as etapas anteriores, ele é primo.
